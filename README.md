## 2. Preparing the TF list

In the FACS data, not all (or most in fact) of the proteins listed are not transcription factors and this requires usage of a lookup table for Mus Musculus (House Mouse) TFs, [data](https://guolab.wchscu.cn/AnimalTFDB4_static/download/TF_list_final/Mus_musculus_TF) found [here](https://guolab.wchscu.cn/AnimalTFDB4/#/Download) and [CITE](https://pubmed.ncbi.nlm.nih.gov/36268869/).
